# Glossary

Use this file for stable project vocabulary.

| Term | Meaning | Notes |
| --- | --- | --- |
| ICM | Interpretable Context Methodology | Folder structure as agent architecture |
| Stage | A numbered workflow folder with one job | Defined under `stages/` |
| Contract | A stage `CONTEXT.md` file | Defines inputs, process, outputs, review, and verification |
| Layer 3 | Stable reference material | Rules, examples, guides, rubrics |
| Layer 4 | Working artifacts | Per-run inputs and outputs |
